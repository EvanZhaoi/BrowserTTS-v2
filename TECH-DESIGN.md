## 1. 概述

### 1.1 项目目标

BrowserTTS v2 是一个浏览器文字转语音的解决方案，提供两个入口：
- **浏览器插件**：快速选中网页文字，一键下载 MP3
- **网页工具**：可调整语速，下载 MP3

### 1.2 核心约束

- **完全离线**：不依赖任何外部 API 或云服务
- **服务端**：运行在服务器的 Docker 容器中
- **输出格式**：MP3，44.1kHz，立体声，128kbps

### 1.3 技术选型

| 组件 | 选型 | 原因 |
|---|---|---|
| TTS 引擎 | Piper TTS | 开源离线，神经网络 |
| 音频转换 | ffmpeg | WAV → MP3 转换 |
| 服务端框架 | Flask | 轻量，和 v1 保持一致 |
| 前端框架 | Vue 3 + Vite | 现代化前端 |
| 容器化 | Docker | 一键部署 |

## 3. 功能设计

### 3.1 语音

| 项目 | 说明 |
|---|---|
| 语音 | 固定中文女声（huayan）|
| 语言 | 中文+英文混合自动处理 |
| 音色 | 单一音色，无需用户选择 |

### 3.2 两个入口

| 入口 | 语速 | 说明 |
|---|---|---|
| 浏览器插件 | 固定（1.0）| 选中文字，一键下载 |
| 网页（Vue3）| 可调（0.5~2.0）| 可调语速下载 |

### 3.3 API 接口

| 接口 | 方法 | 说明 |
|---|---|---|
| `/speak` | POST | 文字转语音，返回 MP3 文件 |
| `/health` | GET | 健康检查 |

#### 3.3.1 POST /speak

**请求：**
```json
{
  "text": "Hello你好World世界",
  "rate": 1.0
}
```

**参数说明：**

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|---|---|---|---|---|
| text | string | ✅ | - | 要转换的文字（建议不超过500字）|
| rate | float | ❌ | 1.0 | 语速（0.5=慢速，2.0=快速）|

**中英文混合处理：**
- 系统自动检测文本中的中/英文段落
- 各自用对应语音模型生成后合并

**响应：**
- 成功：返回 MP3 文件（Content-Type: audio/mpeg）
- 失败：返回错误 JSON

#### 3.3.2 GET /health

**响应：**
```json
{
  "status": "ok",
  "engine": "piper"
}
```

### 3.4 音频输出规格

| 参数 | 值 |
|---|---|
| 格式 | MP3 |
| 采样率 | 44100 Hz（44.1kHz）|
| 声道 | 立体声（Stereo）|
| 比特率 | 128 kbps |

## 4. 核心模块设计

### 4.1 TTS 引擎封装（tts_engine.py）

**功能：**
- 加载 Piper 模型
- 文本分段（中/英文）
- 调用 Piper 生成 WAV
- 合并分段音频
- 语速调整

### 4.2 音频转换封装（audio_converter.py）

**功能：**
- WAV → MP3 转换
- 采样率调整（22050Hz → 44100Hz）
- 立体声转换
- 比特率设置（128kbps）

### 4.3 Flask 服务端（app.py）

**接口：**
- `POST /speak` - 文字转 MP3
- `GET /health` - 健康检查
